This project implements a predictive machine learning model using Scikit-learn to classify email messages as spam or non-spam. The system applies Natural Language Processing (NLP) techniques to convert textual data into numerical features using TF-IDF vectorization. A Multinomial Naive Bayes classifier is then trained on the processed data to predict the category of new email messages.
